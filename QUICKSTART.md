# 🚀 Quick Start - Vodafone Test Analyzer

Get up and running in 2 minutes!

## Step 1: Install (30 seconds)

```bash
cd /Users/ehab.mohamed1/Desktop/script
npm install
```

## Step 2: Start (10 seconds)

```bash
npm start
```

## Step 3: Use (immediate)

1. Open: **http://localhost:3000**
2. Drag & drop your Allure HTML report
3. See instant analysis!

---

## 🎯 What You Get

- ✅ Upload Allure reports via browser
- ✅ Instant visual analysis
- ✅ Pass/fail statistics  
- ✅ Package-level failure grouping
- ✅ Interactive status filters
- ✅ Detailed test information
- ✅ Failure type categorization

---

## 📊 Analysis Includes

- Total/Passed/Failed/Skipped counts
- Pass rate with color indicator
- Execution duration and averages
- Package breakdown
- Common failure reasons
- Failure type distribution
- Top 100 tests with details
- Filter by status (Passed/Failed/Skipped/Broken)
